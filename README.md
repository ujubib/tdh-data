# Workshop: Personal and Research Data Management: Organize, Clean & Enrich

Workshop materials for doctoral students on research data management, organization, and cleaning.

- **Website:** https://ujubib.github.io/tdh-data/  
- **Duration:** 2 hours  
- **Audience:** Doctoral students in Digital Humanities and Cultural Heritage

---

## About This Workshop

This workshop teaches practical skills for managing research data effectively:


- **FAIR principles** for research data
- **File organization** and naming conventions
- **Backup strategies** and storage solutions
- **File structure analysis** with Archifiltre
- **Data cleaning** with OpenRefine

---

## Workshop Contents

- **Slides:** Interactive reveal.js presentation
- **Resources:** Curated guides and tutorials
- **Sample data:** `dirty_data.csv` for hands-on exercises

---

## Tools Covered

- **[Archifiltre](https://archifiltre.fabrique.social.gouv.fr/)** - File structure visualization
- **[OpenRefine](https://openrefine.org/)** - Data cleaning and transformation
- **Storage solutions** - Nuage UPPA, Huma-Num ShareDocs
